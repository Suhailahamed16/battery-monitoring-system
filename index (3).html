Google Sheets Integration Guide

▸ SETUP STEPS

1. Open Google Sheets → Extensions → Apps Script

2. Paste the Web App script (doPost function) to receive data

3. Deploy as Web App → Anyone access → Copy URL

4. Paste URL above in "Google Sheets Web App URL"

5. Arduino → SIM800L sends HTTP POST to the URL with JSON payload

▸ ARDUINO PAYLOAD FORMAT

{"ts":"2025-05-31T10:23:00","b1":12.4,"b2":12.2,"b3":11.8,
 "b4":12.5,"b5":12.3,"b6":12.1,"b7":12.4,"b8":12.2,"b9":12.0,"b10":11.5}

▸ HOSTING (Vercel)

1. Push this HTML to a GitHub repo

2. Import on vercel.com → Deploy (auto detects static site)

3. Update dashboard's SHEETS_URL to your deployed App Script URL
